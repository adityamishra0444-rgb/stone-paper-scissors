# stone-paper-scissors
Overview
This is a command-line Stone Paper Scissor game where a user plays against the computer. The computer randomly selects stone, paper, or scissor, and the game evaluates the winner of each round. Scores are tracked and displayed after the specified number of rounds.
How to Play
	•	Run the Python script.
	•	Enter your name when prompted.
	•	Choose how many rounds you want to play.
	•	For each round, enter your choice number: 1 for Stone 2 for Paper 3 for Scissor
	•	The computer randomly makes its choice each round.
	•	The game shows both choices and declares the round winner.
	•	After all rounds, the final scores and overall winner are displayed.
Rules
	•	Stone beats Scissor
	•	Scissor beats Paper
	•	Paper beats Stone
	•	If both user and computer choose the same, the round is a tie.
Features
	•	Clean console interface that displays choices and scores.
	•	User-friendly prompts and input validation.
	•	Score tracking for both user and system.
	•	Decision logic that handles win, lose, or tie cases correctly.
	•	Plays multiple rounds with final results summary.
